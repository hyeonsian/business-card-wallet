# Business Card Wallet - PRD-lite

## 1) 제품 개요
- 제품명: `Business Card Wallet` (명함 지갑)
- 플랫폼: iOS (Xcode + SwiftUI)
- 목표: 실물 명함을 스캔/OCR하여 디지털 명함으로 저장하고, 검색/정렬/그룹/즐겨찾기로 빠르게 찾을 수 있는 앱 제공

## 2) 문제 정의
- 종이 명함은 보관/검색이 어렵고 분실이 쉽다.
- 연락처를 다시 찾을 때 이름/회사/직함 단위 검색이 불편하다.
- 카테고리별 정리와 빠른 탐색 경험이 부족하다.

## 3) 사용자
- 영업/사업개발/프리랜서/채용 담당자 등 명함을 자주 받는 사용자
- 컨퍼런스/미팅 후 다량의 명함을 빠르게 정리해야 하는 사용자

## 4) 핵심 가치
- 입력 단순화: 스캔 한 번으로 데이터화
- 검색성: OCR 텍스트 기반 즉시 검색
- 정리성: 그룹/정렬/즐겨찾기
- 탐색성: 카드형/리스트형 및 스와이프 중심 UX

## 5) 범위
### In Scope (MVP)
- 명함 촬영/갤러리 선택
- OCR 텍스트 추출 (이름/회사/직함/전화/이메일/주소/기타)
- 명함 저장/수정/삭제
- 그룹 생성/이름 변경/삭제 (기본 그룹 포함)
- 정렬: 최신 등록순(기본), A-Z
- 검색: 전체 텍스트 키워드 검색
- 보기: 카드형, 리스트형
- 즐겨찾기 토글 및 즐겨찾기 필터

### Out of Scope (v2+)
- iCloud 실시간 동기화
- 명함 공유/내보내기(vCard/CSV)
- 고급 페이지 전환(책 넘김 3D), 커스텀 트랜지션 다수
- 다국어 OCR 튜닝/온디바이스 학습

## 6) 주요 사용자 시나리오
1. 사용자는 `+` 버튼으로 명함을 촬영한다.
2. 앱은 자동 크롭/보정 후 OCR 결과를 보여준다.
3. 사용자는 필드를 확인하고 저장한다.
4. 저장 직후 최신순 카드 맨 앞에 반영된다.
5. 사용자는 그룹 선택/검색/정렬/즐겨찾기 필터로 원하는 명함을 찾는다.

## 7) 기능 요구사항
### 7.1 스캔 및 OCR
- 카메라/사진첩 입력 지원
- OCR 비동기 처리 상태 표시 (`분석 중`)
- OCR 실패 시 재시도 및 수동 편집 가능

### 7.2 저장 및 편집
- 명함 이미지 원본 + OCR 필드 저장
- 저장 시 생성일/수정일 기록
- 상세 화면에서 필드 수정 가능

### 7.3 그룹/정렬/필터
- 상단 가로 그룹 탭 + `+ 그룹` 버튼
- `...` 메뉴에서 정렬 전환: `최신순`, `A-Z`
- 그룹 내 `즐겨찾기만 보기` 토글

### 7.4 보기/탐색
- 카드형: 상단 이미지, 하단 텍스트 요약 카드
- 리스트형: 썸네일 + 핵심 텍스트 2~3줄
- 카드형에서 좌/우 스와이프로 이전/다음 탐색

### 7.5 검색
- 이름/회사/직함/전화/이메일/기타 OCR 텍스트 통합 검색
- 현재 선택 그룹 범위 내 검색

## 8) 비기능 요구사항
- 성능: 500장 기준 스크롤/검색 체감 지연 최소화
- 안정성: OCR 실패/권한 거부 시 앱 중단 없이 처리
- 개인정보: 모든 데이터 로컬 저장(MVP)
- 접근성: Dynamic Type, VoiceOver 라벨 기본 대응

## 9) 성공 지표 (MVP)
- 명함 저장 완료율
- OCR 후 수동 수정 비율
- 검색 성공까지 걸리는 시간(사용자 관점)
- 주간 재방문율, 즐겨찾기 사용 비율

## 10) 기술 제안 (MVP)
- UI: SwiftUI
- OCR: Vision (`VNRecognizeTextRequest`)
- 저장소: SwiftData (iOS 17+) 또는 Core Data
- 이미지 저장: 앱 샌드박스 파일 + 경로 참조

## 11) 리스크 및 대응
- OCR 오인식: 수동 편집 UI와 재촬영 플로우 제공
- 다양한 명함 디자인: 필드 추론은 느슨하게, 원문 텍스트 보존
- 스와이프 애니메이션 복잡도: MVP는 기본 전환, 고급 연출은 v2

## 12) 개발 단계 제안
1. Week 1: 데이터 모델 + 저장소 + 기본 목록/상세
2. Week 2: 스캔/OCR + 저장 플로우
3. Week 3: 검색/정렬/그룹/즐겨찾기
4. Week 4: UI polish + 테스트 + 배포 준비
