# Device QA Checklist (iOS)

## 1) 설치/권한
- 앱 첫 실행 시 크래시 없이 진입되는지 확인
- 카메라 권한 허용/거부 각각 동작 확인
- 사진 접근 권한 허용/거부 각각 동작 확인
- 권한 거부 시 `설정 열기` 버튼으로 설정 앱 이동 확인

## 2) 스캔/OCR
- 카메라로 명함 촬영 후 OCR 검수 화면 진입 확인
- 사진첩에서 명함 선택 후 OCR 검수 화면 진입 확인
- OCR 결과에서 전화/이메일/웹사이트 후보 메뉴 노출 확인
- 후보 선택 시 입력 필드가 즉시 변경되는지 확인

## 3) 저장/상세
- OCR 검수 저장 후 홈 목록 즉시 반영 확인
- 카드 탭 시 상세 화면 진입 확인
- 상세 화면 이미지 핀치 확대/더블탭 리셋 확인
- 상세 편집 저장 후 목록 정보 갱신 확인
- 상세 삭제 후 목록에서 제거되는지 확인

## 4) 목록/탐색
- 카드형/리스트형 전환 시 레이아웃 깨짐 없는지 확인
- 즐겨찾기 토글(카드/리스트/상세) 동작 일관성 확인
- 그룹 필터, 검색, 정렬(A-Z/최신순) 조합 동작 확인
- 빈 상태 UI 및 검색 결과 없음 UI 확인

## 5) 회귀 체크
- 스캔 20장 이상 저장 후 스크롤/전환 성능 체감 확인
- 앱 재실행 후 데이터 유지 확인
- 비행기 모드에서도 로컬 기능(조회/검색/편집) 정상 확인
